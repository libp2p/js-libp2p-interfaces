Intro

Installable via npm install --save libp2p-interfaces, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save libp2p-interfaces

You can then require() libp2p-interfaces as normal:

const libp2PInterfaces = require('libp2p-interfaces')

In the Browser

Libp2p-interfaces should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of libp2p-interfaces, including index.js and index.min.js, are included in the /dist folder. Libp2p-interfaces can also be found on unpkg.com under

  • https://unpkg.com/libp2p-interfaces/dist/index.min.js
  • https://unpkg.com/libp2p-interfaces/dist/index.js
